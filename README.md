# Seismic Explorer

**Free, open-source, no-installation earthquake analysis tool**  
Real-time USGS data · Custom polygon filtering · Yearly charts · CSV export  
Works directly inside **Blogger**, WordPress, GitHub Pages and any static website.

Live Demo → https://yourusername.github.io/earthquake-explorer/

![Screenshot](assets/screenshot.png)

## Why This Tool is Unique
- Draw any custom shape on the map (perfect for fault lines, countries, seas, exclusive economic zones)
- All charts, statistics and tables update instantly
- 100 % client-side – no backend, no database, no login
- Fully compatible with **Blogger** (just copy-paste the HTML snippet)
- Trusted USGS real-time data

## How to Add to Your Blogger Blog (2 minutes)
1. Open https://yourusername.github.io/earthquake-explorer/
2. Press Ctrl+A → Ctrl+C (copy everything)
3. In Blogger → Create new post → Switch to **HTML view**
4. Paste (Ctrl+V) → Publish

That’s it – the tool works instantly inside your blog post!

## Features
- Interactive Leaflet map with polygon drawing tool
- Yearly earthquake distribution bar chart (Chart.js)
- Magnitude statistics boxes
- Searchable & sortable DataTables
- One-click CSV export of filtered data
- Mobile-friendly responsive design

## Blogger Compatibility
Tested and confirmed working on all new and classic Blogger templates (2024–2025).  
No external server, no iframe, no script blocking issues.
